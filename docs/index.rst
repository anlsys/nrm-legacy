Welcome to NRM's documentation!
===============================

NRM is an user-space resource management daemon that controls the power usage,
memory boundaries and cpu boundaries of application on HPC systems.

For your first time use of the Node Resource Manager, please see the
:doc:`quickstart <quickstart>` guide.
